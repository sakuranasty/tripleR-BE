# TripleR
## Description
The project is for education about Reducing, Reusing and Recycling

Initial implementation is a little UI showing different kinds of recyclables such as plastic, glass, metal, etc...

The scheme is raw and robably will be corrected

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
